# GroupViz - 可视化策略指南

本文档记录群论可视化的最佳实践和策略，作为开发参考并持续更新。

---

## 1. 可视化设计原则

### 1.1 核心原则

| 原则 | 说明 | 实现方式 |
|------|------|----------|
| **渐进式揭示** | 先展示整体结构，再逐步深入细节 | 分层渲染、动画过渡 |
| **多表征同一性** | 同一概念提供多种表示方式 | Cayley图↔乘法表↔元素列表 |
| **操作可逆性** | 用户操作可undo，保持探索安全感 | 历史栈、撤销功能 |
| **即时反馈** | 每个操作立即显示结果 | 实时计算与渲染 |
| **上下文保持** | 导航时保持当前分析上下文 | 状态持久化 |

### 1.2 认知负荷管理

```
高认知负荷场景          →  解决策略
─────────────────────────────────────────
群运算过程复杂           →  分步动画演示
大量元素同时显示         →  焦点+背景模糊
抽象概念（商群）         →  具体实例+类比
多种颜色同时出现         →  分批展示+图例
```

## 2. 群结构可视化策略

### 2.1 子群 (Subgroups)

**可视化目标**：理解子群作为"群中群"的结构

**策略**：
```
┌─────────────────────────────────────────────────────┐
│  主视图：Cayley图整体                                │
│  ┌───┐    ┌───┐    ┌───┐                          │
│  │e  │────│a  │────│b  │   ← 子群元素用实线圈出     │
│  └───┘    └───┘    └───┘                          │
│                                                     │
│  侧边栏：子群属性                                    │
│  • 阶：3                                           │
│  • 指数：4                                         │
│  • 生成元：{a}                                      │
│  • 是否正规：✓                                     │
└─────────────────────────────────────────────────────┘
```

**关键设计**：
- 子群元素用**实线线圈出**，与周围形成明显边界
- 子群内外用**不同底色**区分（淡入淡出过渡）
- 点击子群元素，高亮其在乘法表中的行/列

### 2.2 陪集 (Cosets)

**可视化目标**：理解陪集是子群的"平移"

**策略**：
```
┌─────────────────────────────────────────────────────┐
│  Lagrange定理演示                                    │
│                                                     │
│  G = {e,a,b,c,d,f}  (|G| = 6)                     │
│                                                     │
│  H = {e,a}     (|H| = 2)                           │
│                                                     │
│  陪集划分动画：                                      │
│  ┌────────┐ ┌────────┐ ┌────────┐                 │
│  │  e  a  │ │  b  d  │ │  c  f  │  ← 3个陪集       │
│  │  (H)   │ │  b+H   │ │  c+H   │                 │
│  └────────┘ └────────┘ └────────┘                 │
│                                                     │
│  验证：|G| = |H| × [G:H] = 2 × 3 = 6 ✓             │
└─────────────────────────────────────────────────────┘
```

**动画序列**：
1. **选择子群H**：高亮子群元素
2. **左乘元素**：逐个选取g∈G，显示gH
3. **颜色编码**：每个陪集用不同颜色
4. **集合动画**：陪集元素移动到对应区域
5. **计数验证**：显示 |H| × [G:H] = |G|

### 2.3 正规子群与商群 (Normal Subgroups & Quotient)

**可视化目标**：理解G/N是"压缩"后的群

**策略**：
```
┌─────────────────────────────────────────────────────┐
│  商群 G/N 构造过程                                    │
│                                                     │
│  原始群 G：D₃ = {e, r, r², s, rs, r²s}             │
│                                                     │
│  正规子群 N = {e, r, r²} ≅ Z₃                       │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │           G 的 Cayley图                      │   │
│  │    (N内元素用蓝色实线圈出)                    │   │
│  │                                             │   │
│  │        r²                                   │   │
│  │       ↙  ↘                                 │   │
│  │     e ←→ r                                 │   │
│  │       ↘  ↙                                 │   │
│  │       s  rs                                │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │           商群 G/N ≅ Z₂                     │   │
│  │                                             │   │
│  │         N     ←→    {s, rs}                 │   │
│  │        (0)    (1)                          │   │
│  └─────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────┘
```

**关键设计**：
- 正规子群用**蓝色虚线框**持续高亮
- 商群元素用**压缩节点**表示（显示为代表元）
- 同余关系用**颜色映射**可视化：g ~ h ↔ 同色

### 2.4 共轭类 (Conjugacy Classes)

**可视化目标**：理解共轭是"坐标系变换"

**策略**：
```
┌─────────────────────────────────────────────────────┐
│  共轭类分解                                          │
│                                                     │
│  S₃ 的共轭类：                                       │
│                                                     │
│  ┌────────────────────────────────────────────┐    │
│  │  类1: {e}           |cl(e)| = 1            │    │
│  │  ┌────┐                                     │    │
│  │  │ e  │  ← 中心元素（单独高亮）              │    │
│  │  └────┘                                     │    │
│  ├────────────────────────────────────────────┤    │
│  │  类2: {(12), (13), (23)}  |cl| = 3        │    │
│  │  ┌────┐  ┌────┐  ┌────┐                    │    │
│  │  │(12)│  │(13)│  │(23)│  ← 3-换位同色      │    │
│  │  └────┘  └────┘  └────┘                    │    │
│  ├────────────────────────────────────────────┤    │
│  │  类3: {(123), (132)}   |cl| = 2            │    │
│  │  ┌────┐  ┌────┐                             │    │
│  │  │123 │  │132 │  ← 3-循环同色               │    │
│  │  └────┘  └────┘                             │    │
│  └────────────────────────────────────────────┘    │
│                                                     │
│  类方程：|S₃| = 1 + 3 + 2 = 6                       │
└─────────────────────────────────────────────────────┘
```

**交互设计**：
- 选择元素a，显示所有gag⁻¹的计算过程
- 拖动一个元素，展示其如何"移动"到共轭位置

## 3. 群运算可视化策略

### 3.1 Cayley乘法表

**可视化目标**：快速查询运算结果，发现群结构模式

**策略**：
```
┌─────────────────────────────────────────────────────┐
│  D₃ 乘法表                                           │
│       │  e   r  r²  s  rs r²s                      │
│  ─────┼─────────────────────────────────           │
│    e  │  e   r  r²  s  rs r²s                      │
│    r  │  r  r²   e r²s   s  rs                     │
│   r²  │ r²   e   r  rs r²s   s                     │
│    s  │  s  rs r²s   e  r  r²                      │
│   rs  │ rs r²s   s  r²  e   r                      │
│  r²s  │r²s   s  rs   r r²   e                      │
│  ─────┴─────────────────────────────────           │
│                                                     │
│  [高亮模式]                                          │
│  ┌────────────────────────────────────────────┐    │
│  │  点击 r 行 → 高亮整行，显示 r·x = ?         │    │
│  │  点击 s 列 → 高亮整列，显示 x·s = ?         │    │
│  │  r·s = rs (交叉格高亮)                      │    │
│  └────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────┘
```

**交互功能**：
- **行探索**：点击某行，高亮该元素左乘的所有结果
- **列探索**：点击某列，高亮该元素右乘的所有结果
- **模式发现**：自动检测并标记对称性（阿贝尔群对角线对称）
- **逆元标记**：每个元素的逆元用特殊边框标记

### 3.2 运算律验证动画

**结合律 (a·b)·c = a·(b·c)**
```
步骤1：显示 (a·b) 的计算路径
        a ──→ ab ──→ (ab)c
               ↓
              结果

步骤2：显示 a·(b·c) 的计算路径  
        c ←── bc ←── a(bc)
                      ↓
                     结果

步骤3：两条路径到达同一点，动画闪烁确认
```

**交换律**（仅阿贝尔群）
```
自动检测：若表关于主对角线对称
        → 显示"交换律成立"徽章
        → 允许拖动交换因子顺序
```

## 4. 定理可视化策略

### 4.1 Lagrange定理

```
┌─────────────────────────────────────────────────────┐
│  Lagrange定理：|H| 整除 |G|，|G| = |H| × [G:H]     │
│                                                     │
│  1. 选择子群 H ⊆ G                                   │
│                                                     │
│     G = {0,1,2,3,4,5}  (Z₆)                        │
│     H = {0,3}         (2阶子群)                     │
│                                                     │
│  2. 动态生成左陪集                                  │
│                                                     │
│     0+H = {0,3}        ← 蓝色                       │
│     1+H = {1,4}        ← 红色                       │
│     2+H = {2,5}        ← 绿色                       │
│                                                     │
│  3. 可视化计数                                      │
│                                                     │
│     ┌────────────────────────────────────────┐      │
│     │  |G| = 6 = |H| × [G:H]                 │      │
│     │      ↓       ↓      ↓                  │      │
│     │      6   =   2  ×   3   ✓             │      │
│     └────────────────────────────────────────┘      │
│                                                     │
│  4. 推广：任意子群阶必整除群阶                       │
└─────────────────────────────────────────────────────┘
```

### 4.2 同构定理

**第一同构定理可视化**：
```
┌─────────────────────────────────────────────────────┐
│  第一同构定理：G/ker(φ) ≅ im(φ)                     │
│                                                     │
│  示例：φ: Z₆ → Z₃,  φ([x]) = [x] mod 3            │
│                                                     │
│  核：ker(φ) = {0,3} ≅ Z₂                           │
│                                                     │
│  ┌──────────────┐    φ    ┌──────────────┐        │
│  │     G        │ ──────→ │    im(φ)      │        │
│  │  Z₆ = {0-5}  │         │    Z₃ = {0-2} │        │
│  │              │         │              │        │
│  │  [0,3]≡0    │         │      0       │        │
│  │  [1,4]≡1    │         │      1       │        │
│  │  [2,5]≡2    │         │      2       │        │
│  └──────────────┘         └──────────────┘        │
│        ↓ 商群                    ↑ 同构映射        │
│  ┌──────────────┐    ≅      ┌──────────────┐        │
│  │   Z₆/{0,3}   │ ──────→ │     Z₃      │        │
│  │  = {0,1,2}   │         │              │        │
│  └──────────────┘         └──────────────┘        │
│                                                     │
│  验证：|G/ker| = 6/2 = 3 = |im| ✓                 │
└─────────────────────────────────────────────────────┘
```

**可视化要点**：
- 并排展示核、像、商群
- 用相同颜色标记对应元素
- 动画演示压缩过程

### 4.3 轨道-稳定子定理

```
┌─────────────────────────────────────────────────────┐
│  轨道-稳定子定理：|G| = |Orb(x)| × |Stab(x)|        │
│                                                     │
│  示例：D₄ 在顶点 {1,2,3,4} 上的作用                 │
│                                                     │
│  群 G = D₄ = {e, r, r², r³, s, rs, r²s, r³s}      │
│                                                     │
│  选取 x = 1                                        │
│                                                     │
│  稳定子 Stab(1) = {e, s}  (|Stab| = 2)            │
│                                                     │
│  轨道 Orb(1) = {1,2,3,4}  (|Orb| = 4)            │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │           顶点排列                            │   │
│  │              1                                │   │
│  │           ↙   ↘                              │   │
│  │         4       2                            │   │
│  │           ↘   ↙                              │   │
│  │              3                                │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  验证：|G| = 8 = |Orb| × |Stab| = 4 × 2 = 8 ✓     │
└─────────────────────────────────────────────────────┘
```

## 6. 布局策略

### 6.1 Cayley图布局

| 群类型 | 推荐布局 | 说明 |
|--------|----------|------|
| 循环群 Zₙ | 圆形 | 元素沿圆周排列，生成元指向下一节点 |
| 二面群 Dₙ | 双环 | 内圈旋转r，外圈反射s |
| 对称群 Sₙ | 力导向 | 自动布局，保持对称性 |
| 阿贝尔群 | 网格 | 行最简形式展示 |

### 6.2 圆圈图平面布局

对于具有共享元素的循环子群（如Q8、直积群），采用分层布局减少边交叉：

| 层级 | 位置 | 元素类型 |
|------|------|----------|
| 核心 | 圆心 | 单位元 e |
| 外层 | 大圆周 | 共享元素（多个循环子群共有） |
| 内层 | 小圆周 | 特定循环子群的独有元素 |

**算法要点**：
- 检测所有共享元素的循环子群集合
- 共享元素沿外层圆周均匀分布
- 各循环子群的独有元素沿生成元方向从共享点向外扇形展开
- 大幅减少边交叉，提升平面性

### 6.3 多视图协调

```
┌─────────────────────────────────────────────────────┐
│  主视图 (Cayley图)     │  侧边栏                    │
│                        │  ┌─────────────────────┐   │
│      ○ ○ ○ ○          │  │ 乘法表              │   │
│      ○ ○ ○ ○          │  │ e  a  b  c         │   │
│      ○ ○ ○ ○          │  │ a  b  c  e  ←高亮  │   │
│      ○ ○ ○ ○          │  │ b  c  e  a         │   │
│                        │  │ c  e  a  b         │   │
│                        │  └─────────────────────┘   │
│                        │  ┌─────────────────────┐   │
│                        │  │ 元素属性             │   │
│                        │  │ a                   │   │
│                        │  │ 阶: 4               │   │

│                        │  │ 共轭类: {...}       │   │
│                        │  └─────────────────────┘   │
└─────────────────────────────────────────────────────┘
```

## 7. 动画设计

### 7.1 动画类型

| 场景 | 动画效果 | 时长 |
|------|----------|------|
| 元素运算 | 沿路径移动 + 缩放 | 800ms |
| 陪集生成 | 元素渐次高亮 | 500ms/元素 |
| 商群压缩 | 缩小组合并移入新节点 | 1200ms |
| 共轭变换 | 对称翻转效果 | 1000ms |
| 定理验证 | 数字递增动画 | 600ms |

### 7.2 动画控制

- **播放/暂停**：用户可控动画进程
- **速度调节**：0.5x, 1x, 2x, 4x
- **步骤回退**：支持单步回退
- **自动播放**：循环演示

## 9. 辅助功能

### 9.1 数学符号渲染

使用 KaTeX 渲染所有数学公式，替代 Unicode 纯文本：
```
Z₆, D₄, S₃, A₄    →  TeX: Z_6, D_4, S_3, A_4
V₄, Q₈             →  TeX: V_4, Q_8
H ⊆ G, N ⊲ G       →  TeX: H \subseteq G, N \triangleleft G
G/N, ker(φ), im(φ) →  TeX: G/N, \ker(\varphi), \operatorname{im}(\varphi)
≅, →, ∈, ∀, ∃      →  TeX: \cong, \to, \in, \forall, \exists
```

KaTeX 渲染覆盖所有位置：2D/3D节点标签、面板内容、图例、提示信息。

### 9.2 自适应复杂度

```
初学者模式：
  • 显示所有标签和说明
  • 限制可选群的范围
  • 提供逐步引导

进阶模式：
  • 隐藏部分提示
  • 允许自定义群
  • 显示抽象结构

专家模式：
  • 纯符号界面
```

## 10. 性能优化

### 10.1 渲染优化

- **大型群（S₄以上）**：使用 Canvas 而非 SVG
- **大量节点**：使用 WebGL (Three.js) 渲染
- **懒加载**：只渲染可视区域内的元素

### 10.2 计算优化

- **缓存**：子群、陪集等计算结果缓存
- **Web Worker**：复杂计算移至后台线程
- **渐进计算**：先显示占位符，后填充结果

## 12. 已实现功能

### 12.1 S₃ 对称群可视化

**Cayley图**
- 圆形布局，6个节点均匀分布
- 生成元 σ₁₂ (红色) / σ₂₃ (青色) 带箭头曲线边
- 节点可点击选择，高亮当前选中元素
- 支持hover效果，显示节点详情

**Cayley乘法表**
- 6×6 交互式表格
- 点击单元格查看运算结果
- hover高亮对应结果

**键盘导航**
- ←/→ 键切换选中元素
- 实时更新侧边栏信息

**动画效果**
- 节点和边的过渡动画
- 动画速度可调节 (0.25x - 2x)

### 12.2 多群支持

- 循环群 Zₙ (n=1..20)：圆形布局
- 二面群 Dₙ (n=3..8)：双环布局
- 交错群 Aₙ (n=3..5)：力导向布局
- S₄ (24阶)：截角立方体/菱形截角八面体等多形状3D模板
- A₅ (60阶)：截角二十面体/截角十二面体3D模板
- V₄ (Klein四群)、Q₈ (四元数群)
- 直积群：Z₄×Z₂、Z₂³、Z₃×Z₃ (晶格布局)

### 12.3 子群格视图 (SubgroupLatticeView)

- Hasse图表示子群的包含关系
- 节点按层级排列（平凡子群在底部）
- 正规子群用紫色高亮
- 边表示直接包含关系
- 支持节点点击查看子群详情

### 12.4 对称性视图 (SymmetryView)

- 多面体几何体 (正四面体、立方体、正八面体、正二十面体、正十二面体)
- 群元素映射为几何旋转变换
- 双击元素触发动画演示旋转操作
- 旋转轴与交点标记可视化
- 轴方向从几何数据运行时计算，无需硬编码

### 12.5 多视图浮动窗口

- 支持同时打开多个浮动窗口
- 每个窗口可显示不同视图模式
- 窗口可拖拽、调整大小
- 主画布与浮动窗口联动（共享同一群状态）

### 12.6 子集保存与分析

- 选中元素集保存为子集（8色调色板）
- 自动检测乘法封闭性（子群判定）
- 自动检测共轭封闭性（正规子群判定）
- 可管理多个子集

### 12.7 国际化 (i18n)

- 支持中文 / English 切换
- 所有UI文本、提示信息、操作历史均翻译
- 语言偏好持久化到 localStorage

### 12.8 小群预计算注册表

- 阶<12的所有群（共19个）自动索引
- 预计算子群、正规子群、共轭类、中心数据
- 懒加载机制，首次访问时计算
